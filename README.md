# Умная сверка 3.0 (TicketProf ↔ Bars Tour)

Программа для автоматического сопоставления номенклатуры продаж (TicketProf) и приходов (Bars Tour) из Excel-файлов, вычисления маржинальности и подготовки отчетов (Excel и Word Акт сверки).

## 🚀 Как скачать и запустить программу на новом ПК

Выполните следующие шаги:

### 1. Скачивание проекта

**Вариант А: Через Git (рекомендуется)**
Откройте командную строку (CMD или PowerShell) в папке, куда хотите скачать проект, и выполните:
```bash
git clone https://github.com/ardont/barsik.git
cd barsik
```

**Вариант Б: Без Git (просто архив)**
1. Перейдите по ссылке: [https://github.com/ardont/barsik](https://github.com/ardont/barsik)
2. Нажмите зеленую кнопку **Code** ➔ **Download ZIP**.
3. Распакуйте скачанный архив в удобную папку и перейдите в нее.

---

### 2. Запуск программы

Для работы программы нужен только установленный Python (рекомендуется версия 3.10 или 3.11).

1. Дважды кликните по файлу **`setup.bat`**:
   - Он автоматически создаст изолированную виртуальную папку `.venv`;
   - Установит все необходимые библиотеки (`customtkinter`, `pandas`, `openpyxl`, `python-docx`).
   *Этот шаг нужно сделать только один раз при первом запуске.*

2. Дважды кликните по файлу **`run.bat`**:
   - Программа запустится в красивом графическом интерфейсе.

---

## 🛠 Краткая инструкция по работе с интерфейсом

1. В верхней строке нажмите кнопку **«Обзор...»** и выберите ваш файл Excel со сверкой (например, `08.07_1.xlsx`).
2. Нажмите кнопку **«▶ Сверить данные»**.
   - Программа за несколько секунд проведет сопоставление по сложным формулам и идентификаторам билетов/отелей.
   - Заполнятся KPI карточки (общая прибыль, процент совпадений, сумма ошибок).
3. Во вкладке **«Все сопоставления»** вы увидите сопоставленные строки:
   - **Зеленые строки** — сопоставление прошло успешно.
   - **Оранжевые строки** — записи есть только в одной системе (TicketProf или Bars Tour).
   - **Красные строки** — записи сопоставились, но маржа отеля не равна 10% (нетипичная маржа).
4. Кликните по любой строке в таблице, чтобы в нижней панели **«Детали»** отобразилась подробная расшифровка по операции.
5. Во вкладках **«Только в TicketProf»** и **«Только в Bars Tour»** собраны все расхождения.
6. На вкладке **«Ручное сопоставление»**:
   - Вы можете выбрать одну строку слева (из TicketProf) и одну строку справа (из Bars Tour), а затем нажать кнопку **«⛓ Связать»**.
   - Программа моментально пересчитает сверку и запомнит эту связь на будущее (сохранит в `data/manual_links.json`).
7. Кнопки **«Экспорт Excel»** и **«Экспорт Word»** сохранят готовые отчеты в ту же папку, где лежит исходный файл.
